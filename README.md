# website-development-

Этот проект содержит сайт с кнопкой "Курсы" и переходом на страницу курсов.

## Как запустить

1. Склонируйте репозиторий:
   ```
   git clone https://github.com/Adrian23217/website-development-.git
   ```
2. Установите зависимости:
   ```
   npm install
   ```
3. Запустите проект:
   ```
   npm start
   ```
